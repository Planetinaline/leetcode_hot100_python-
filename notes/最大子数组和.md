# 最大子数组和

给定一个数值数组nums，返回其最大子数组和

思路：
本题使用动态规划，定义子问题为，以nums[i]结尾的连续子数组的最大和为多少？
dp[i]表示该问题的解
则dp[0] = nums[0]
dp[n] = dp[n - 1] + nums[n] if dp[n - 1] >=0 else nums[n] n = range(1, len(nums))

```python
size = len(nums)
if size == 0:
    return 0

dp = [0] * size
dp[0] = nums[0]
for i in range(1, size):
    if dp[i - 1] >= 0:
        dp[i] = dp[i - 1] + nums[i]
    else:
        dp[i] = nums[i]
return max(dp)
