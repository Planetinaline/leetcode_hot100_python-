# 旋转图像

给定一个n * n的二维矩阵matrix，要求将图像顺时间旋转90度。
要求必须原地旋转图像。

思路：
竖着看：
第一列元素去了第一行
第二列去了第二行
第j列去了第j行

横着看：
第一行元素去了最后一行
第二行元素去了倒数第二行
第i行元素去了第n - 1 - i行 i从0开始

结果：
第i行第j列的元素去了第j行第n - 1 - i列
即(i, j) -> (j, n - 1 - i)
注意到：(i, j) -> (j, i) -> (j, n - 1 - i)
即顺时针转90度等价于将矩阵先转置后翻转行
则有以下实现代码：
```python
n = len(matrix) # n为行数
for i in range(n):
    for j in range(i): 
        # 枚举矩阵下半三角元素
        # matrix[j][i] = matrix[i][j]
        # 交换需要写为
        matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]

for row in matrix:
    row.reverse()
```

如果枚举矩阵上半三角，可以在一个循环内完成：
```python
n = len(matrix)
for i, row in enumerate(matrix):
    for j in range(i + 1, n):
        matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
    row.reverse()
```
