# 相交链表

给定两个单链表的头节点headA与headB，请你找出并返回两个单链表相交的起始节点。如果不存在相交节点，则返回null。
题目保证整个链式结构中不含有环。
函数返回结果后，链表需要保持其原始结构。

思路：
设两条链表相交前的长度分别为x, y，相交后的公共长度为z。
当x == y时，p = headA，q = headB，同时出发必定会到达相交处。
当x != y时，注意到x + z + y == y + z + x，即p = headA出发遍历链表，如果遇到空节点则令p = headB；同理，q = headB出发遍历链表，如果遇到空节点则令q = headA，如果p, q同步出发，则必定会到达相交处。
当两条链表没有相交时，由于链表的结尾都是空节点，则可视为相交于空。
由于x + y == y + x，则此时若p, q同时出发，则同样可以到达相交处。
代码如下：
```python
p, q = headA, headB
while p is not q:
    p = p.next if p else headB
    q = q.next if q else headA
return q
```
