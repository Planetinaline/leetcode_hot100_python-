# 找到字符串中所有异位词

给定一个字符串数组，请将其中的所有异位词分组，可以按任意顺序返回结果列表。(所有的异位词中都只包含小写字母)

hashtable中，key是唯一的，value是不唯一的。正如此题，异位词的顺序是不同的（或者说，是彼此不同的字符串），但是其包含的字母，以及各个字母的个数是一样的。
这道题的难点就是设计区别不同异位词的key。
```python
hashtable = {}
for st in strs:
    key = [0] * 26
    for ch in st:
        key[ord(ch) - ord('a')] += 1
    if tuple(key) not in hashtable:
        hashtable[tuple(key)] = [st]
    else:
        hashtable[tuple(key)].append(st)
return [val for val in hashtable.values()]
```

简洁写法：
```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        d = defaultdict(list)  # 如果 key 不在字典中，则自动插入一个空列表
        for s in strs:
            sorted_s = ''.join(sorted(s))  # 把 s 排序，作为哈希表的 key
            d[sorted_s].append(s)  # 排序后相同的字符串分到同一组
        return list(d.values())  # 哈希表的 value 保存分组后的结果
```
