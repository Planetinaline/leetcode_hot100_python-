# 除了自身以外数组的乘积

给定一个整数数组nums，要求返回answer数组，其中answer[i]是nums数组中除了nums[i]以外的其他元素的乘积。
题目保证乘积不会整型溢出，要求不使用除法，且算法的时间复杂度在o(n)。

思路：
如果允许使用除法，则先计算nums数组中全部元素的乘积，再遍历数组将乘积依次除以元素即可。
如果使用暴力解法，时间复杂度为o(n^2)
故采取空间换时间的做法。
作如下定义：
pre[i]为nums数组中nums[i]之前所有元素，即nums[0]到nums[i - 1]的乘积。
那么有，
pre[i] = nums[i - 1] * nums[i - 2] * ... * nums[0] 
pre[i] = pre[i - 1] * nums[i - 1]
同理有，
suf[i]为nums数组中nums[i]之后的所有元素，即nums[i + 1]到nums[n - 1]的乘积，n = len(nums)
suf[i] = nums[i + 1] * nums[i + 2] * ... * nums[n - 1]
suf[i] = suf[i + 1] * nums[i + 1]
为了计算方便，定义pre[0] = suf[n - 1] = 1
则有answer[i] = pre[i] * suf[i], i = range(0, n)
```python
n = len(nums)
pre = [1] * n
for i in range(1, n):
    pre[i] = pre[i - 1] * nums[i - 1]

suf = [1] * n
for i in range(n - 2, -1, -1):
    suf[i] = suf[i + 1] * nums[i + 1]

return [p * s for p, s in zip(pre, suf)]
