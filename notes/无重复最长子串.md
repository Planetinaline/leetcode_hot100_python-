# 无重复最长子串

给定一个字符串s，返回其中最长无重复子串的长度。

思路：本题使用滑动窗口思想。
该思想的核心有两个：
1、 遍历的i是窗口的尾部（因为如果是头部，则仍需要向前遍历）
2、 使用的数据结构是队列

附上代码：
```python 
from collections import deque
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        # sliding window
        l = len(s)
        if l == 0:
            return 0
        max_len = 0
        window = deque()
        win_tail = 0
        cur_len = 0
        while win_tail < l:
            if s[win_tail] not in window:
                window.append(s[win_tail])
                cur_len += 1
                win_tail += 1
                max_len = max(max_len, cur_len)
            else:
                window.popleft()
                cur_len -= 1
        return max_len
