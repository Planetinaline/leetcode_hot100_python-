# 螺旋矩阵

给定一个m行n列的矩阵matrix，请按照顺时针螺旋顺序，返回矩阵中所有元素。

思路：
从(0, 0)位置出发，按“右下左上”顺序依次访问矩阵，当碰到矩阵边界或者已访问的元素时调转方向。
```python
DIRS = (0, 1), (1, 0), (0, -1), (-1, 0)
m, n = len(matrix), len(matrix[0])
i = j = di = 0 # 当前方向为DIRS[di]，初始化为右
ans = [] # 结果列表
for _ in range(m * n):
    ans.append(matrix[i][j])
    matrix[i][j] = None # 对于已经访问过的元素，将其置为None
    x, y = i + DIRS[di][0], DIRS[di][1] # 需要判断的是下一步的位置 ！！！！！！！
    if x < 0 or x >= m or y < 0 or y >= n or matrix[x][y] == None:
    # if i < 0 or i >= m or j < 0 or j >= n or matrix[i][j] == None: # 判断是否需要转向
        di = (di + 1) % 4
    i += DIRS[di][0]
    j += DIRS[di][1]
