# 移动零

给定一个数值数组，需要写一个函数，把数组中所有0元素放置到数组末尾，同时保持非零元素的相对位置。需要inplace操作。

思路：
比如说[1, 0, 3, 0, 0, 12, 0, 9]
是否可以使用两个连续的指针i, j遍历数组：
1、当nums[i] == 0 and nums[j] == 0，j++ 
2、当nums[i] == 0 and nums[j] != 0, swap(i, j), i++, j++
3、当nums[i] != 0 and nums[j] == 0, i++
4、当nums[i] != = and nums[j] != 0, i++, j++

```python
for i, n in enumerate(nums):
    # 从左往右，找第一个0的位置
    if n != 0:
        continue
    tmp = i
    # 以第一个0为起点，找第一个不为0的位置
    for j in range(i + 1, len(nums)):
        if nums[j] != 0:
            tmp = j
            break
    # 交换两者的值
    nums[i], nums[tmp] = nums[tmp], nums[i]

题解思路：
使用双指针，左指针指向非零序列的尾部，右指针向右探索寻找非零元素
每当右指针找到一个非零元素，则交换左右指针元素值
```python
n = len(nums)
left = right = 0
while(right < n):
    if nums[right] > 0:
        nums[left], nums[right] = nums[right], nums[left]
        left += 1
    right += 1
