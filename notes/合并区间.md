# 合并区间

给定一个区间数组intervels，合并其中所有能合并的区间，返回一个没有重叠的区间数组。

思路：
例[[1, 2], [2, 4], [4, 5]] -> [[1, 5]]
又例[[3, 5], [3, 4], [9, 10], [1, 3]] -> [[1, 5], [9, 10]]

首先对intervels中的intervel进行按照左端点大小的排序
[[1, 3], [3, 5], [3, 4], [9, 10]]
然后创建一个栈
ans = []
枚举intervel:
for intervel in intervels:
    # 入栈
    if len(ans) > 0 and intervel[0] <=  ans[-1][1]:
        # ans[-1][0] = intervel[0]
        ans[-1][1] = max(intervel[1], ans[-1][1])
    else:
        ans.append(intervel)
