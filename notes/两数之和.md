# 两数之和

输入为非排序的数值数组a，和一个target数值。
且满足：
2 <= length(a) <= 1e4，
-1e9 <= nums[i] <= 1e9，
-1e9 <= target <= 1e9，
要求输出i,j，满足nums[i] + nums[j] == target，且 i != j。

首先想到的是枚举<i, j>，通过。
然后是双指针和哈希的做法：
1、 双指针


2、 哈希
核心思想，存储遍历过的元素，把遍历过的元素加入到集合中
创建一个hashtable = dict()
其中的key和value分别是nums[i]和i
对nums数组进行遍历，对于每一个元素num：
    如果target - num的key在hashtable中，则返回key对应的value，即hashtable[target - num]，即为j
    如果不在，则把hashtable[num] = i 加入到hashtable中

由于本题的本质是在数组中寻找一对下标(i, j)，使得nums[i] + nums[j] == target，则首先想到的就是枚举(i, j)，这样做的时复为o(n^2)，空复为o(1)。
由于内存假设不存在瓶颈，则现在的想法是尽可能设计时间复杂度低的算法。
综上，考虑使用空间换时间的做法，即使用哈希表存储遍历过的元素。
```python
hashtable = {}
for i, x in enumerate(nums): # 遍历数组
    if target - x not in hashtable:
        hashtable[x] = i
    else:
        return [hashtable[target - x], i]
```
