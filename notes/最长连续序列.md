# 最长连续序列

给定一个非排序的数值数组，要求返回该数组中最长的连续序列的长度（不要求在原数组中连续）。
比如[5, 1, 2, 3, 100, 101, 200]，返回3。
要求时间复杂度为o(n)。

思路：
以[5, 1, 2, 3, 100, 101, 200]为例，
要找到最长连续序列，首先要从数组中找到一般的连续序列。
该数组中共包含4个连续序列：[1, 2, 3], [5], [100, 101], [200]，
寻找的方式是：
1、首先找到序列的开端x
2、查询x + 1是否在数组中
为达到o(n)的时间复杂度，需要把查询操作的时间复杂度控制在o(1)。
pseudo code:
```python
# turn nums into a hash table in order to do o(1) search
u_set = set(nums)
max_len = 0
for num in nums:
    if num - 1 in u_set:
        continue
    length = 1
    while num + length in u_set:
        length += 1
    max_len = max(max_len, length)
return max_len

这一版没有通过所有用例，因为有些case中会有很多重复的开头，比如0，0，0。
u_set = set(nums)
pass_set = set()
max_len = 0
for num in nums:
    if num - 1 in u_set:
        continue
    if num in pass_set:
        continue
    length = 1
    while num + length in u_set:
        length += 1
    max_len = max(max_len, length)
    pass_set.add(num)
return max_len

更优解是直接遍历set
u_set = set(nums)
max_len = 0
for n in u_set:
    if n - 1 in u_set:
        continue
    length = 1
    while n + length in u_set:
        length += 1
    max_len = max(max_len, length)
return max_len
```

20260422 review:

此题的关键在于找序列起点，对于数组中任意一个元素x，判断x是否为起点的做法是，判断x - 1是否在数组中。
因为时间复杂度的限制，需要哈希表保证查询的时复为o(1)
故：
```python
u_set = set(nums) # 将nums放入到set以保证查询到时复为o(1)
max_len = 0
for n in u_set:
    if n - 1 in u_set:
        continue
    # 此时n一定为开端
    tmp = n + 1
    while tmp in u_set:
        tmp += 1
    max_len = max(max_len, tmp - n)
    if max_len * 2 >= len(u_set):
        break
return max_len
```
